### Installation

1. Get a free API Key at [https://developers.themoviedb.org/3](https://developers.themoviedb.org/3)

2. Install NPM packages

```sh
npm install
```

3. Enter your API in `.env`

```JS
API_KEY = 'ENTER YOUR API';
```

##### Development Mode

```shell
npm run dev
```

##### Production Mode

```shell
npm run build
```

##### Liver Server 

```shell
npm run start
```
